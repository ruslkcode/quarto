# Project 99 – Quarto

## Description
Client–server implementation of the board game Quarto.
The project was developed as part of the Software Systems course.

## Features
- Client-server architecture
- Human vs Human gameplay
- Human vs AI gameplay
- Text-based user interface (TUI)
- Multiple AI strategies

## How to Run
1. Start the server
2. Run the client
3. Choose Human or AI mode
4. Queue for a game

## Controls
- queue – join matchmaking
- rank – show ranking
- quit – exit the client

## Technologies
- Java
- TCP networking
- Multithreading
